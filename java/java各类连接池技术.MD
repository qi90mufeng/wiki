pool （管理维持对象）



池的功能

1、初始化，预先放入对象

2、设定最大数，防无尽连接

3、出借对象出池

4、归还对象回池



场景需求（运营）

1、管理redis连接

2、200并发数

3、redis pool最大10

4、服务对象维护



产品原型（需求）

1、初始化

​	最初创建3个连接对象，最大数10个

2、销毁

​	关闭池子，不再出借对象

3、取连接

​	1、有空闲连接，则返回一个

​	2、没有空闲连接，未达到上限，创建一个返回

​	3、没有空闲连接，已达到上限，等待其他归还

4、释放连接

​	对象归还后，释放连接
